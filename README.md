# Sales-Performance-Analytics-Dashboard

## Project Overview

This project focuses on analyzing sales data for  a  Hardware Store, a brick-and-mortar business dealing in hardware products. The goal of this project is to transform raw sales data into meaningful insights using **SQL for data analysis** and **Power BI for interactive dashboards**.

The project helps business stakeholders understand sales performance, identify profitable products, and make data-driven decisions to improve revenue and operational efficiency.

---

## Problem Statement

Hardware Store faced challenges in analyzing and understanding their sales data. Due to the lack of proper analytics tools and dashboards, it was difficult for management to:

- Track overall sales performance
- Identify top performing products
- Understand customer demand patterns
- Make strategic business decisions

This project solves the problem by building a **data analysis pipeline and interactive dashboards** that provide clear and actionable insights.

---

## Technologies Used

- **SQL / MySQL** – Data storage and querying
- **Power BI** – Data visualization and dashboard creation
- **Power Query** – Data cleaning and transformation
- **DAX (Data Analysis Expressions)** – Calculated measures and KPIs

---


## Dashboard Features

The Power BI report includes three main dashboards:

### 1. Key Insights Dashboard
Provides a high-level overview of sales performance including:

- Total revenue
- Total sales quantity
- Top customers
- Revenue by market

### 2. Profit Analysis Dashboard
Analyzes profitability of products and markets:

- Profit margin by product
- Revenue vs profit trends
- Cost analysis

### 3. Performance Insights Dashboard
Tracks sales performance across time and markets:

- Sales growth trends
- Market contribution
- Customer segmentation

---

## Key Business Insights

Using the dashboard, stakeholders can:

- Identify **top performing markets and products**
- Monitor **sales trends over time**
- Understand **customer purchase behavior**
- Optimize **pricing and inventory strategies**
